Combination number 2

2020-05-13

数据复杂度在1e10时，不能再使用此前打表迭代的做法。比较直观的想法是从表达式入手，$ C_a^b = \frac{a!}{(a-b)! * b!} $, 用两个表来记录 fact的值，和逆元fact的值即可将时间复杂度变为NlogN。
